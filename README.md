# HR-Analytics-Project
This HR Analytics project focuses on analyzing employee data to generate meaningful insights that help organizations improve workforce management and HR decision-making. The dataset contains employee-related information such as department, job role, salary, education, gender, performance rating, and attrition status.
The project begins with data preprocessing and transformation, where missing values, duplicate records, and inconsistencies were handled to improve data quality. SQL queries were used to analyze employee distribution, department-wise performance, salary trends, and attrition patterns. Excel was used for data preparation and preliminary analysis.

Interactive dashboards were created in Power BI and Tableau to visualize important HR metrics including employee attrition rate, gender diversity, department-wise employee count, job satisfaction, salary distribution, and workforce demographics. The dashboards provide clear and user-friendly visualizations that support data-driven HR decisions.

Through this analysis, key insights were identified regarding employee turnover, workforce distribution, and performance trends across departments. The project helps HR teams understand which factors contribute to employee attrition and highlights areas where employee engagement and retention strategies can be improved.
